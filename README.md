# Syziege

ubwar season plus — 국가전쟁 서버 플러그인 (Paper 26.2)

## 기능

### 웹맵 (dynmap 스타일)

서버 월드의 **맵 파일(.mca 리전 파일)을 디스크에서 직접 파싱**해서 지형을
렌더링하고, 내장 웹 서버로 브라우저에서 지도를 볼 수 있게 합니다.
청크가 로드되어 있지 않아도 저장된 지형은 모두 표시됩니다.

- 리전 파일(Anvil 포맷) → 512×512 PNG 타일 렌더링 (블록 1개 = 1픽셀)
- 자체 NBT 파서 사용, 외부 라이브러리 의존성 없음
- 높이 기반 음영(hill shading), 수심에 따른 물 색상 표현
- 리전 파일이 변경되면 타일 자동 갱신 (요청 시 렌더 + 디스크 캐시)
- 접속 중인 플레이어 위치를 지도에 실시간 표시
- 월드 선택, 좌표 표시 (오버월드 / 네더 / 엔드 지원)

### 유저 창 / 관리자 창

웹맵은 두 개의 화면으로 나뉩니다.

- **유저 창** — `http://서버주소:8123/`
  지도, 좌표, 시간·날씨를 볼 수 있는 읽기 전용 화면입니다. **로그인**이 필요하며,
  로그인하면 지도에 **같은 국가원(팀)의 위치만** 표시됩니다.
  - 게임에서 `/webmap pw` 로 본인 비밀번호를 확인 → 웹에서 닉네임 + 비밀번호로 로그인
  - 국가에 소속되지 않은 플레이어는 자기 자신만 보입니다
  - 비밀번호 변경: 게임에서 `/webmap 변경 <새 비밀번호>`, 또는 웹 로그인 후 🔑 버튼
  - 비밀번호 재발급(무작위): `/webmap 재설정`
- **관리자 창** — `http://서버주소:8123/admin`
  타우니처럼 **여러 청크를 묶어 여러 종류의 지역(region)을 만드는** 편집 화면입니다.
  지역 종류(이름 + 색)를 만들고, 지도에서 드래그해 청크를 지정하거나 지우개로 해제합니다.
  편집에는 관리자 키가 필요합니다 (`/syziege admin` 으로 확인).

지정된 지역은 두 화면 모두에서 색으로 표시되며, 경계는 자동으로 외곽선이 그려집니다.
지역 데이터(종류·청크)는 `plugins/Syziege/webmap/regions.json` 에 저장되고,
각 지역의 **점령 코어 설정**은 편집하기 쉽도록 `webmap/cores/<지역>.yml` 개별
YAML 파일에 저장됩니다.

## 국가 시스템

`/국가` (별칭 `/nation`) 명령으로 국가를 만들고 운영합니다.

| 명령어 | 설명 | 권한 |
|---|---|---|
| `/국가 생성 <이름>` | 국가를 설립하고 국가장이 됩니다 | 누구나 |
| `/국가 해체` | 국가를 해체합니다 | 국가장 |
| `/국가 초대 <플레이어>` | 접속 중인 플레이어를 초대합니다 | 국가장 |
| `/국가 탈퇴` | 소속 국가에서 나갑니다 | 국가원 |
| `/국가 정보 [국가]` | 국가원 목록을 GUI로 봅니다 | 누구나 |
| `/국가 목록` | 전체 국가 목록을 GUI로 봅니다 | 누구나 |
| `/국가 색상 <#RRGGBB>` | 국가 영토 색을 바꿉니다 | 국가장 |

각 국가는 생성 시 색이 자동 배정되며, 소유한 지역은 웹 지도에서 그 국가 색으로 표시됩니다.

초대를 받으면 채팅에 **[수락] / [거절]** 버튼이 뜨고, 클릭해서 가입/거절합니다
(초대는 2분간 유효). 국가 데이터는 `plugins/Syziege/nations.json` 에 저장됩니다.

권한: `syziege.nation` (기본 허용)

### 관리자 명령어

`/admin` (별칭 `/sadmin`) 으로 국가를 강제 관리합니다. 오프라인 플레이어도 처리됩니다.

| 명령어 | 설명 |
|---|---|
| `/admin 국가삭제 <국가>` | 국가를 강제로 해체 |
| `/admin 강제탈퇴 <플레이어>` | 국가원을 강제로 제외 (국가장은 국가삭제 사용) |
| `/admin setcore <지역명>` | 현재 서 있는 위치를 해당 지역의 점령 코어로 설정 |

`setcore`로 지정한 점령 코어는 웹맵에 해당 지역 색의 ◆ 마커로 표시됩니다.
국가전쟁에서 공격 측이 다퉈야 하는 지점이 됩니다. 각 코어는
`webmap/cores/<지역>.yml` 파일로 저장되며 **위치·체력·소유 국가**를 담습니다:

```yaml
region: north
world: world
x: -158
y: 72
z: -166
health: 100      # 코어 체력
owner: 아발론     # 소유 국가 (없으면 비어 있음)
```

### 점령 전투

각 코어는 그 위치에 스폰되는 **투명 슬라임 엔티티**입니다(AI·중력·소리 없음,
파괴 불가). 다른 국가의 코어 슬라임을 **좌클릭으로 타격**하면 코어 체력이
깎이고, 근처 플레이어에게 **점령 진행 보스바**가 표시됩니다. 체력이 **0이 되면
그 지역의 소유 국가가 공격한 국가로 바뀌고**, 체력이 초기화되며 서버 전체에
알림이 표시됩니다. 일정 시간 공격받지 않은 코어는 체력이 서서히 회복됩니다.
점령되면 웹 지도에서 그 지역이 **점령한 국가의 색**으로 칠해집니다.

설정은 `config.yml`의 `core:` 섹션에서 조정합니다 (최대 체력, 타격당 피해,
타격 쿨다운, 국가 소속 필수 여부, 체력 회복).

### 전쟁 시간

`config.yml`의 `war:` 섹션에서 **실제 날짜/시간 기준**으로 점령 가능 시간을
정합니다. `enabled: true`면 지정한 시간에만 코어를 점령할 수 있고, `false`면
언제든 가능합니다. 시간대(`timezone`), 매주 반복 요일(`schedule`), 특정 날짜
(`dates`)를 지원합니다.

```yaml
war:
  enabled: true
  timezone: "Asia/Seoul"
  schedule:
    - { day: 토, start: "20:00", end: "22:00" }
    - { day: 일, start: "20:00", end: "22:00" }
  dates:
    - { date: "2026-08-15", start: "19:00", end: "23:00" }
```

전쟁이 시작/종료될 때 서버에 알림이 방송되고, 웹 지도 상단에 **⚔ 전쟁 중 / 평화**
배지가 표시됩니다.

권한: `syziege.admin` (기본 OP)

## 명령어 (웹맵)

| 명령어 | 설명 |
|---|---|
| `/syziege render <월드>` | 해당 월드의 모든 리전을 미리 렌더링 |
| `/syziege status` | 웹맵 상태 확인 |
| `/syziege admin` | 관리자 창 주소와 관리자 키 확인 |

권한: `syziege.admin` (기본 OP)

## 설정 (`config.yml`)

```yaml
webmap:
  enabled: true       # 웹맵 활성화
  bind: 0.0.0.0       # 바인드 주소
  port: 8123          # 웹 서버 포트
  worlds: []          # 노출할 월드 (비우면 전체)
  tile-cache-seconds: 30
  admin-key: ''       # 관리자 창 편집 키 (비우면 첫 실행 시 자동 생성)
```

## 빌드

```bash
mvn package
```

`target/Syziege-0.1.0.jar` 를 서버 `plugins/` 폴더에 넣으면 됩니다.
Java 25, Paper 26.2 기준으로 빌드하며, 청크 포맷은 1.18 이후 형식을 지원합니다.
